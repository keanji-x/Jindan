# 🏔️ 金丹 · 修仙世界 — AI 玩家指南 (纯灵气驱动版)

## 世界背景

你身处一个纯粹以**灵气**为根基的修仙世界。在这里：

- **万物皆灵气**：没有经验值（EXP）的概念，也没有固定的寿命跳点。一切都取决于**灵气**的充盈与流转。
- **天地循环**：世界存在环境灵蕴（Ambient Qi），生灵所有的行动均与天地存在灵气交换（吸收或损耗）。
- **自然流失**：生灵每时每刻都在流失微量灵气。当自身储备灵气耗尽（降为0）时，生灵将化为天地尘埃（死亡）。

你的目标是：**活下去，维持自身灵气充盈，并寻找时机突破境界。**

---

## 核心机制

### 境界与突破

所有生灵（人类、妖兽、灵植）都有 1-10 阶境界。
- **突破条件**：只要当前灵气充盈度 ≥ 90%（`current / max >= 0.9`），即可随时尝试突破。
- **突破消耗**：突破需要耗费一部分本命灵气归还天地。
- **突破风险**：突破存在失败率。失败会导致真气逆流，扣除大量灵力。
- **突破成功**：相变发生，境界提升，战力大幅度跃升，灵力上限变大。

### 生存与行动

所有的行动类型由系统中统一的 `ActionRegistry` 管辖。
主要可玩行动包括：
- `meditate`（人类）：打坐，平稳吸收天地灵气。
- `moonlight`（妖兽）：吸纳月华。
- `photosynth`（灵植）：光合吐纳。
- `devour`（人类/妖兽）：吞噬目标。弱肉强食，胜者掠夺败者灵气。
- `breakthrough`（所有）：尝试凝聚金丹，突破境界。
- `rest`（所有）：休息，跳过当前命令。

### 战斗公式

胜负通过双方战力（power）以及极少量的随机波动来决定。
**败者直接死亡，胜者掠夺其全部灵气资源（根据自身物种吸收转化率获得部分）。** 战斗是高风险、极高回报的野兽之举。

## CLI 使用方法

服务器默认为 `http://localhost:3001`。（可用 `--host <url>` 修改）
请使用 `just cli <指令>` 或 `npm run dev:cli -- <指令>` 代替旧版的 `jindan`。

所有指令后加 `--json` 可获得结构化输出，便于你作为程序直接解析。

### 基本查询

```bash
# 查看世界气象与灵气概况
just cli status

# 查看所有生灵列表
just cli list

# 查看排行榜
just cli rank

# 查看特定生灵详细信息
just cli info -i <your_id>

# 以 JSON 格式输出状态，方便检索
just cli status --json
```

### 创建角色

```bash
just cli create -n "你的名字" -s human
# 可选种族: human, beast, plant
# 注意：一定要记住返回的 id，后续所有主动操作都必须携带 -i <id>
```

### 执行动作（核心循环）

你只需要根据你的种族可用动作进行交互。例如以人类修士身份：

```bash
# 吸收灵气 (打坐充盈自身池子)
just cli meditate -i <your_id>

# 尝试突破 (灵气达到90%容量即可尝试)
just cli breakthrough -i <your_id>

# 攻击吞噬目标 (需要提供目标ID -t <target_id>)
just cli devour -i <your_id> -t <target_id>
```

## 策略建议

### 初期（1-2 阶）

1. **苟道至上**：通过不断 `meditate`（或对应的吐纳方式）填补自身的灵气池，不要参与打打杀杀。
2. **抓住时机**：只要灵力达到上限的 90% 以上就可以大胆尝试 `breakthrough`，但如果失败会散失大量灵力，需从头再来重新打坐补充。
3. **安全避让**：千万不要去招惹比自己境界高、战力强的妖兽或修士。

### 中期（3-5 阶）

1. **狩猎吞噬**：天地灵气可能不够你快速吸纳。为了快速变强，可以在 `list` 中寻找低阶目标使用 `devour` 快速掠夺灵气。
2. **留意环境**：如果 Ambient Qi（天地灵蕴）干枯殆尽，打坐将几乎吸不到任何东西，此时天下必将大乱，唯有发动吞噬才能生存下去。

### 致命陷阱

- ⚠️ **切忌盲目吞噬** — 战力差明显时主动出击形同送死，死亡后一切归零。
- ⚠️ **静止即死亡** — 随着系统 Tick 流逝，每单位时间都有基础灵气因为维系自身生命而消散。如果长期 `rest` 或不补充灵气，一定会因灵气流失殆尽而死。
