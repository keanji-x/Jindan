# 金丹世界 · 背景设定

## 天地初开

太古之初，混沌未分。一道先天灵气贯穿虚空，化为万物之基。灵气凝为山川、散为风雨，最终沉入地脉深处，形成了**灵脉**——天地灵气的根源。

灵脉如同世界的心脏，日夜不息地向天地间吐纳灵气。这股弥漫在空气中、渗透在土壤里的无形能量，被称为**天地灵气**。

当某处灵气过于浓郁、超出天地所能承载的极限时，灵气便会自然凝结为固态——**灵石**。灵石晶莹剔透，内蕴纯粹灵力，是修行界通用的硬通货。

## 修士

凡人之中，偶有天赋异禀者，能感知天地灵气的流动。他们尝试将灵气引入体内，淬炼经脉、壮大灵力——这便是**修炼**的起点。

修士的成长以**境界**划分。每一次突破，都是对自身极限的挑战：灵力需要充盈到临界点，修为积累必须深厚，还需要一分运气和悟性。境界越高，寿命越长，战力越强，但所需的修炼资源也呈指数增长。

修士之道，说到底只有一条——**争**。争灵气、争灵石、争机缘。灵气有限，修士无穷，弱者被淘汰，强者通吃一切。

## 妖兽

灵气孕育万物，不止人类。山中走兽、水中游鱼，久沐灵气之后也会开启灵智，化为**妖兽**。

妖兽不修功法，不炼丹药，靠本能吸收天地灵气壮大己身。它们体内会凝结**妖丹**——这是妖兽一生灵力的结晶，对修士而言价值连城。

于是修士与妖兽之间形成了一条食物链：修士猎杀妖兽获取妖丹和灵石，妖兽的存在又消耗着区域灵气，间接影响修士的修炼效率。

## 生死循环

这个世界最残酷也最公平的法则是——**万物终将归于天地**。

无论多强大的修士，寿元总有耗尽之时。坐化之际，体内蕴含的灵力重新弥散回天地，生前积攒的灵石散落尘世，供后来者拾取。妖兽亦然——寿终正寝或被斩杀，灵力回归，妖丹留与胜者。

正是这种生死循环，维持着世界的资源平衡：

- 修士太多 → 灵气稀薄 → 修炼困难 → 弱者淘汰
- 修士太少 → 灵气充沛 → 灵石遍地 → 新人崛起
- 灵脉枯竭 → 灵气断绝 → **末法降临**

## 末法之忧

灵脉虽强，储量终究有限。每一天的吐纳，都在消耗灵脉的根基。

当灵脉储量耗尽，天地灵气将不再补充。修士无法修炼，妖兽饥寒而亡，灵石成为最后的稀缺资源。

这便是修仙界最大的隐患——**末法时代**。

在末法降临之前，谁能攀上境界巅峰，谁就能笑到最后。

## 世界规则摘要

| 机制 | 说明 |
|------|------|
| **灵脉** | 每天向世界注入灵气 (500/天)，总储量有限 (100,000) |
| **灵气** | 区域能量池 (上限 5,000)，修炼和妖兽都会消耗 |
| **灵石** | 灵气溢出凝结 (10:1)，死亡掉落，通用货币 |
| **修炼** | 吸收 50 灵气 → 获得 ~10 经验 |
| **突破** | 经验达标 + 灵力充足 → 70-90% 成功率升级 |
| **战斗** | 战力决定胜率 (sigmoid)，败者死亡 |
| **寿命** | 100 + 阶数×50 天，超龄即坐化 |
| **妖兽** | 被动消耗灵气，死后掉妖丹，自动刷新 |
